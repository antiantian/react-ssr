### 目录结构

```bash
├── /build              # 服务端打包文件夹
├── /client             # 客户端入口
├── /public             # 客户端打包文件夹
├── /server             # 服务端入口
├── /src                # 项目源码目录 
├── package.json        # 项目信息
├── webpack.client.js   # 客户端webpack配置
└── webpack.server.js   # 服务器webpack配置
```
 

### 快速开始

克隆项目文件:
 
进入目录安装依赖:

```bash
 
 npm i 或者 npm install
```

 

```bash
 
npm run dev:server 服务端打包
npm run dev:start  启动项目
npm run dev:client 客户端打包 
 
打开 http://localhost:9093
```
